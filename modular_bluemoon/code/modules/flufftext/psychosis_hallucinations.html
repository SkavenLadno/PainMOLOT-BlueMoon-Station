modular_bluemoon/code/modules/flufftext/psychosis_hallucinations.dm

/proc/build_psychosis_tier_poolsОдин раз за раунд раскладывает GLOB.psychosis_hallucination_list по трём tier-пулам. Lazy-build вызывается из status_effect/psychosis.
/proc/psychosis_overlay_otherНакладывает image на живого моба в view target. Возвращает image для контроля жизненного цикла, либо null если кандидатов нет.
/proc/distort_messageПодменяет 1-2 совпадения из словаря в исходной строке. Если совпадений нет - возвращает оригинал без изменений. Поиск регистронезависимый и токеновый: "ok" не сработает внутри "broken", "да" - внутри "дать".
/proc/psychosis_word_regexРегекс с word-boundary для distort_message. Граница - всё, что не латинская или кириллическая буква и не цифра. Конструируется на каждый вызов, потому что регексы в DM stateful и кеширование требует ручного сброса next.

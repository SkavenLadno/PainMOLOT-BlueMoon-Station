Space Station 13 - Modules - Types — Var Details - Proc Details

pda
/obj/item/modular_computer/pda

Modular PDA

A handheld computer that replaces the legacy /obj/item/modular_computer/pda system. Uses the modular_computer framework with TGUI-based programs.

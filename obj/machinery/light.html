Vars

individual_switch_stateLight can be connected to its individual light switch by tapping it with light switch frame. This variable changes during flipping individual switch. If this variable is: null -> light works as usual. TRUE/FALSE -> area.lightswitch will be ingored and this variable will be checked instead.

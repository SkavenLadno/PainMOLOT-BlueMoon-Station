tools/Sprite_Grayscale_Converter/readme.md

А если серьёзно, это конвертер специально для превращения grayscale спрайтов в rgb спрайты и наоборот

Если очень просто как этим пользоваться

  1. Компилим SS13IconsConverter.dme
  2. Открываем SS13IconsConverter.dmb
  3. Выбираем режим (изначально стоит RGB to Greayscale)
  4. Выбираем .dmi файл
  5. получаем готовый .dmi

Известные проблемы

  1. Часть dmi файлов не читается вовсе (помолитесь богу-машине я хз)
  2. При превращении из RGB в Grayscale генерируется всегда 3 слоя независимо от наличия оных
  3. При превращении из RGB d Grayscale любые спрайты с переходными цветами (те, у которых RGB не чёткая и имеет значения других цветов) приводит к тому, что цвет остаётся у двух слоёв
  4. Чисто чёрный (0,0,0) удаляется при превращении из RGB d Grayscale

ATMOS_MACHINE_IDLE_HEARTBEAT

While idle, vents/scrubbers only run a full recheck this often. Event wakes (turf activation, pipenet pressure jump, radio signals, power, welding) clear the idle state instantly, so this is a safety net, not the response time.

Procs

GetItemSchedules a reference scan for a GC-failed datum. Returns the qdel_item datum for a type path or type-path string, or null if none exists yet.
GetOrCreateItemReturns the qdel_item datum for a type, creating it if it doesn't exist yet.
GetQueuedDatumReturns the datum currently represented by the queue slot, or null if the slot is stale.
HandleLevelProcess all pending expired entries at the given queue level. Uses tombstoning + head-pointer advancement — no static vars, safe across runtimes.
HardDeleteForce-delete an object that failed to GC gracefully. Separated into its own proc for profiling clarity.
MaybeCompactCompact the dead prefix of a queue level if enough tombstoned entries have accumulated.
OnLevelFailHandles a GC failure at the given queue level — logs, notifies, and escalates.
QueueEnqueue a datum for GC checking at the given level.
RecoverQueueEntriesImports queued entries while preserving both original qdel time and current stage time.
gc_notify_opted_adminsSends a GC notification message only to admins who have opted into leak notifications.
